Разработка web-приложения «База данных резюме».

Используемые технологии: Java, Intellij IDEA, Git, Maven, Log4j2, PostgreSQL, JUnit5, JDBC, Apache Tomcat, HTTP, Servlets,JSP/JSTL, XML, JSON. Хранение резюме: в памяти на основе массива, отсортированного массива, списка и ассоциированного массива(Map), в файловой системе, в стандартной и кастомной сериализации Java, в формате JSON (Google Gson), в формате XML (JAXB), в реляционной базе PostgreSQL. Деплой веб-приложения в контейнер сервлетов Apache Tomcat.
